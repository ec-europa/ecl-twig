# ECL Twig - EC Accordion

npm package: `@ecl-twig/ec-component-accordion`

```shell
npm install --save @ecl-twig/ec-component-accordion
```

## Accordion

### Parameters

- "extra_classes" (string) (default: '')
- "extra_attributes" (array) (default: []): format:
  - "name" (string) (default: '')
  - "value" (string) (default: '')
