# ECL Twig - EC Button component

npm package: `@ecl-twig/ec-component-button`

```shell
npm install --save @ecl-twig/ec-component-button
```

## Primary button

### Parameters

- "label" (string) (default: '')

### Example:

```twig
{% include 'path/to/button.html.twig' with {
  variant: 'primary',
  label: 'Example button'
} %}
```
